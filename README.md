# KodON Science Club Project for Hack4Sages Hackathon
## Running script
1. Set up python environment (only the first time):
    ```bash
    sh set_up.sh
    ```
2. Launch app:
    ```
    sh run.sh
    ```
3. Go to http://localhost:8080 or http://127.0.0.1:8080 to start using app.
